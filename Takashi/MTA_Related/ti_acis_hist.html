ACIS Histgram Trending

Last Updated: Sep 30, 2014

Description

This set of scripts extracts ACIS histgram mode data from archive and creates trends plots.

Data Needed

  • /data/mta_www/mta_temp/mta_states/MJ//comprehensive_data_summary
  • /data/mta/Script/ACIS/Acis_hist/Data/ccd_node_
  • We also need output from arc4gl.

Environment Setting

  • These scripts run under SOT python2.7 environment. Set:
    setenv PYTHONPATH "/proj/sot/ska/arch/x86_64-linux_CentOS-5/lib/python2.7/site-packages"

Output Locations and File Names

  • /data/mta/Script/ACIS/Acis_hist/Data/
    • ccd#ccd#_node#node#_#pos#, e.g., ccd3_node0_low
  • /data/mta/Script/ACIS/Acis_hist/Data/Dist_data/CCD#ccd#/node#node#/#pos#/
    • hist_#year#_#mm# e.g., hist_2013_09
  • /data/mta_www/mta_acis_hist/Plot_trend/CCD#ccd#/
    • count_rate_high_trend.png
    • count_rate_low_trend.png
    • peak_position_high_trend.png
    • peak_position_low_trend.png
    • peak_width_high_trend.png
    • peak_width_low_trend.png
