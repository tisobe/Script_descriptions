How to Run the Script(s)

  1. go to: cd /data/mta/Script/Sib/Correct_excess/
  2. set: setascds
  3. run: /usr/local/bin/perl Sib_corr/sib_corr_find_observation.perl <start> <stop>
    where start and stop are the time of the interval start and stop time in the format of mm/dd/yy,hh:mm:ss, e.g. 01/01/14;00:00:00
    this will create a file acis_obs
  4. go to :cd /data/mta/Script/Sib/Correct_excess/Lev2
  5. copy: cp ../acis_obs .
  6. run: wap_script
  7. reset env: rm -rf param; mkdir param;source /home/mta/bin/reset_param
  8. run: /usr/local/bin/perl ./correct_factor.perl
  9. run: /usr/local/bin/perl ./find_excess_file.perl
  10. run: /usr/local/bin/perl /data/mta/Script/ACIS/SIB/Correct_excess/Sib_corr/sib_corr_comb.perl <start> <end>
    where <start> <end> is start and stop time in the format of <yyyy>:<ydate><hh>:<mm>:<ss>, e.g., 2014:213:00:00:00
  11. move: mv /data/mta/Script/ACIS/SIB/Correct_excess/Lev2/Data/* /data/mta/Script/ACIS/SIB/Lev2/Data/Data_<year>_<month>/.
  12. go to :cd /data/mta/Script/Sib/Correct_excess/Lev1
  13. copy: cp ../acis_obs .
  14. run: wap_script
  15. reset env: rm -rf param; mkdir param;source /home/mta/bin/reset_param
  16. run: /usr/local/bin/perl ./correct_factor.perl
  17. run: /usr/local/bin/perl ./move_exess_list.perl
  18. run: /usr/local/bin/perl /data/mta/Script/ACIS/SIB/Correct_excess/Sib_corr/sib_corr_comb.perl <start> <end>
    where <start> <end> is start and stop time in the format of <yyyy>:<ydate><hh>:<mm>:<ss>, e.g., 2014:213:00:00:00
  19. move: mv /data/mta/Script/ACIS/SIB/Correct_excess/Lev1/Data/* /data/mta/Script/ACIS/SIB/Data/Data_<year>_<month>/.
  20. go to : cd /data/mta/Script/Sib/Correct_excess/
  21. set: setenv PYTHONPATH "/proj/sot/ska/arch/x86_64-linux_CentOS-5/lib/python2.7/site-packages"
  22. run: /data/mta/Script/ACIS/SIB/Correct_excess/Scripts/ccd_comb_plot.py
  23. run: /data/mta/Script/ACIS/SIB/Correct_excess/Scripts/update_html.py
  24. run: /data/mta/Script/ACIS/SIB/Correct_excess/Scripts/ccd_comb_plot.py lev2
  25. run: /data/mta/Script/ACIS/SIB/Correct_excess/Scripts/update_html.py lev2
