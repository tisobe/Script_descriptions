How to Run the Script(s)

  1. Go to: cd /data/mta/Script/Month/SIM/
  2. Set: setenv PYTHONPATH "/proj/sot/ska/arch/x86_64-linux_CentOS-5/lib/python2.7/site-packages"
  3. Run: create_monthly_sim_plots
  4. Move: cp *png /data/mta4/www/REPORTS/MONTHLY/#year##MMM#/

How to Run Test(s)

  1. Go to: cd /data/mta/Script/Month/SIM/
  2. Set: setenv PYTHONPATH "/proj/sot/ska/arch/x86_64-linux_CentOS-5/lib/python2.7/site-packages"
  3. Run: ./Scripts/grating_plot.py
  4. Run: ./Scripts/sim_plot.py
  5. Run: ./Scripts/aorwspd_plot.py

You will see something like:

If the unit test fails, you will see where the problem happened.
