Creating Orbital Data RDB files

Last Updated: Sep 24, 2014

Description

This set of scripts extract orbital element related data from archived data and create rdb files.

Data Needed

  • orbitf*_eph1.fits, anglef*_eph1.fits using arc4gl.

Environment Setting

  • ASCDS (source /home/mta/.ascrc)
  • source /home/mta/bin/reset_param

Output Locations and File Names

  • /data/mta/Script/Orbital/Orb/
    • orbit_data_<time stamp>
    • Data_<year>/orbit_data_<time stamp>
  • /data/mta/DataSeeker/data/repository/
    • aorbital.rdb

How to Run the Script(s)

  1. go to: cd /data/mta/Script/Orbital/Exec
  2. run: /data/mta/MTA/bin/orb_elm_wrap_script

How to Run Test(s)

  1. go to: cd /data/mta/Script/Orbital/Exec
  2. run: /data/mta/MTA/bin/orb_elm_wrap_script_test

Expected result from the test

The results are save in /data/mta/Script/Orbital/Test_out. Compare the results to those in /data/mta/Script/Orbital/Orbital/Test_comp_results/.
