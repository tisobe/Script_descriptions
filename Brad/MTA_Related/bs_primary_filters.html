How to Run the Script(s)

  1. go to: cd /data/mta/Script/Dumps/Scripts/
  2. set:
    setascds
    setenv ASCDS_CONFIG_MTA_DATA "/home/ascds/DS.release/config/mta/data"
    setenv IPCL_DIR "/home/ascds/DS.release/config/tp_template/P011/"
    setenv ACORN_GUI "/home/ascds/DS.release/config/mta/acorn/scripts/"
    setenv LD_LIBRARY_PATH "/home/ascds/DS.release/lib:/home/ascds/DS.release/ots/lib:/soft/SYBASE_OSRV15.5/OCS-15_0/lib:/home/ascds/DS.release/otslib:/opt/X11R6/lib:/usr/lib64/alliance/lib"
  3. copy: cp ./house_keeping/msids.list ./house_keeping/otg-msids.list .
  4. run: /data/mta/Script/Dumps/Scripts/run_filter_scripts.py
  5. move: mv -f *.tl /data/mta/Script/Dumps/
  6. run: /data/mta/Script/Dumps/Scripts/otg_proc.pl
  7. run: /data/mta/www/mta_sim/Scripts/getdata
  8. cat data: ./gratstat.lis >> /data/mta/Script/Dumps/gratstat.lis

Second Method: getting data from archive.

  1. go to: cd /data/mta/Script/Dumps/
  2. run arc4gl to extract data.
    • operation = retrieve
    • dataset = flight
    • detector = telem
    • level = raw
    • tstart = <start time > (e.g., 12/01/02)
    • tstop = <end time > (e.g., 01/01/03)
    • go
  3. remove *log* file
  4. unzip the file: gzip -d *sto.gz
  5. create input list: ls *.sto > xtmpnew
  6. run: /home/ascds/DS.release/bin/acorn -nOC $workdir/otg-msids.list -f<file name>
  7. run: otg_proc.pl
  8. clean: rm *.sto *.tl
