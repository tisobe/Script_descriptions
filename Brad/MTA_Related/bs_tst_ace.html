This directory contains a couple of perl scripts to monitor the ACE satellite proton fluxes for Chandra operations, plus the associated data files. This is a part of TST related collections of scripts.

Please read /data/mta4/proj/rac/ops/README and /data/mta4/proj/rac/ops/ACE/README file for more details.
